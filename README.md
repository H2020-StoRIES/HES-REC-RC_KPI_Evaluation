
# KPI Evaluation Module

This module calculates key performance indicators (KPIs) for each scenario after the Simulink simulation and optimization stages. It processes simulation results and dispatch data to evaluate system-level metrics such as efficiency, flexibility, emissions, and operational cost.

---

## 📁 Files

- `KPI_evaluation.py`  
  Main script for processing output data and calculating KPIs. It reads result files and writes KPI outputs in both `.json` and `.xlsx` formats.

- `Metric_Calculator1.py`  
  Contains the core functions and logic for computing various KPIs, such as:
  - Electrical and thermal efficiency
  - Flexibility factors
  - Levelized cost of energy (LCOE)
  - Operational cost and emissions

---

## 📊 KPI Summary

After all runs are processed, the module creates a `KPI_summary.xlsx` file containing:
- `Sheet1`: Individual KPI results for each scenario and run
- `Average`: Mean KPI values across scenario for each run

---

## 🔎 Notes

- The input files must be generated by the simulation and pipeline process and must follow the expected naming and structure.
- KPI calculations depend on consistent units and well-defined profiles from previous steps.
